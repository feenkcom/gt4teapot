# gt4teapot
[Glamorous Toolkit](https://github.com/feenkcom/gtoolkit) for [Teapot](https://github.com/zeroflag/Teapot). Teapot is a micro web framework for [Pharo](http://pharo.org).

## How to load

Execute the following code in a [Glamorous Toolkit image](https://github.com/feenkcom/gtoolkit):
```
Metacello new
   baseline: 'GToolkit4P3';
   repository: 'github://feenkcom/gt4p3';
   load.
```
